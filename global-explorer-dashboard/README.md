# Global Explorer Dashboard

A React application that allows users to explore countries, their weather, and local news using external APIs.

## Features

- **Country Explorer**: Browse all countries with search, filter, and pagination
- **Country Details**: View comprehensive country information including:
  - Basic details (capital, population, region, etc.)
  - Current weather in the capital city
  - Latest news headlines from the country
  - Interactive map location
- **Favorites System**: Mark countries as favorites with persistent storage
- **Dark/Light Mode**: Toggle between themes
- **Responsive Design**: Works on all device sizes

## APIs Used

1. **REST Countries API** - Country data and flags
2. **OpenWeatherMap API** - Weather data
3. **NewsAPI** - Latest news headlines
